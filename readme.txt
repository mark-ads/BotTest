Бот для помощи риелторам. В боте собраны различные Ассистенты от Open Ai (аналоги GPTs, но со своим API).
Здесь представлена небольшая часть файлов и функций.
Всего зарегистрировано порядка 50 риелторов.

Бот работает на серверах Яндекс Клауд.
Работает самодельный мониторинг (отдельный бот запущенный на сервере).
Реализован автоматический деплой изменений через GitHub Actions. Можно сказать, что это CI/CD, нажал кнопку в IDE с пушем и через 30 секунд все необходимые файлы поменялись на сервере. 
Изменения сначала затрагивают тестового бота (идентичный клон основного бота), а затем, после успешных тестов меняется основной бот.

Краткий функционал бота:

  Система ролей (админ, пользователь)
  Авторизация каждого действия
  Логирование
  Общение небольшого кластера ботов через Kafka (немного необходимых микросервисов)
  Преобразование аудио в текст через Yandex Speechkit
  Обращение к Open AI через API
  Изменение настроек ассистентов изнутри самого телеграмм-бота

Вы можете подключиться боту (тестовая версия), чтобы посмотреть на фукнционал для пользователя. Администраторский доступ могу выдать отдельно.
